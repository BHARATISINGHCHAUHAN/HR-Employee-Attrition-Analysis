# HR Employee Attrition Analysis using SQL

## Project Overview
This project analyzes the IBM HR Employee Attrition dataset using MySQL. 
The objective is to explore employee attrition patterns and generate meaningful business insights using SQL.

## Dataset
- IBM HR Analytics Employee Attrition Dataset
- Total Records: 1,470

## Tools Used
- MySQL Workbench
- SQL
- Excel

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- CASE WHEN
- Subqueries
- CTE (Common Table Expressions)
- Window Functions (ROW_NUMBER, RANK)
- 40+sql queries

## Key Business Questions
- What is the overall employee attrition rate?
- Which department has the highest
-attrition?
- Which job roles have the highest attrition?
- How does overtime affect attrition?
- Does business travel impact attrition?
- How does job satisfaction relate to    attrition?
- Which department have the highest average monthly income?

## Key Insights
- Total Employees: 1,470
- Overall Attrition Rate: 16.12%
- Sales department had the highest attrition rate.
- Employees who travel frequently showed higher attrition.
- Employees working overtime were more likely to leave.
- Single employees had the highest attrition rate.
- The sales department had the highest average monthly income among the three departments analyzed
